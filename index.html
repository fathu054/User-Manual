1. Getting Started

Welcome to KNT SteelWorks, your all-in-one solution for managing steelworks business operations. This app is designed to streamline worker management, contractor tracking, work logging, and financial reporting.

Before You Begin

For the best experience, we recommend completing these steps in order:

  1. Add all your workers and their details
  2. Add contractors you work with
  3. Start logging daily works
  4. Regularly check reports for business insights

App Sections Overview

Works Dashboard

View and manage daily work entries, filter by date, and add new works.

Reports

Access financial summaries, salary reports, contract analytics, and export data.

Note Pad

Keep personal notes, site instructions, and client feedback organized.

Manage Data

Manage workers, contractors, works, and search through all records.

5. Adding & Tracking Works

Work entries are the core of the application. Each work entry records a job done for a contractor.

Work Management Screenshots

Add New Menu

Quick access to add different types of entries

Add Work Form

Detailed form to log new work entries

Creating a Work Entry

Access Works Section

Go to Works from the bottom navigation or Manage â†’ Works.

Add New Work

Tap Add Work button (visible if no works exist for selected date).

Fill Work Details

Complete all fields:

  • Work Date: Date when work was performed
  • Contractor: Select from your contractor list
  • Site Name & Location: Where work was done
  • Work Description: Details of the work performed
  • Assigned Workers: Select workers assigned to this job
  • Materials Used: List of materials (optional)
  • Amount: Contract value or payment amount

Save Work Entry

Tap SAVE WORK to log the entry. It will now appear in your works list and reports.

Viewing and Searching Works

There are multiple ways to find work entries:

  • Date Filter: Use the date picker in the Works dashboard
  • Search: Use the search bar in the Manage Data section
  • Contractor Filter: Filter works by contractor in Reports section

7. Note Pad

The Note Pad is your digital notebook for keeping track of ideas, reminders, site instructions, and client feedback.

Note Pad Screenshot

The Note Pad interface for creating and managing notes

Creating a Note

  1. Go to Notes from the bottom navigation
  2. Tap + New Note (or + Create First Note if no notes exist)
  3. Enter your note title and content
  4. Add tags if needed (e.g., #important, #site-visit)
  5. Tap save to store the note

Note Organization

Use these features to keep notes organized:

  • Search: Find notes by keywords
  • Tags: Categorize notes for easy filtering
  • Date Stamping: Notes automatically record creation date
  • Edit/Delete: Modify or remove notes as needed
Use Case: Create notes for site-specific instructions that workers need to follow. You can reference these notes when assigning works.

9. Tips & Best Practices

Complete Profiles

Fill in all details for workers and contractors. Complete profiles help in communication and reporting.

Daily Updates

Update work entries daily to maintain accurate records and reports.

Regular Backups

Export important reports monthly for offline backup and accounting purposes.

Use Search

Leverage the search function in Manage Data to quickly find entries instead of scrolling.
